# Audio Chatbot Project

This repository is for learning and developing an **audio chatbot**.  
It will include experiments, mini-projects, and the main project as it evolves.

The project aims to explore:

- Audio input/output in the browser
- Speech-to-text (STT)
- Text-to-speech (TTS)
- Real-time chat interactions with a bot

More details and documentation will be added as the project progresses.
